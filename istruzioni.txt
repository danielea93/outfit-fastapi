Installare python, ricordandoti di spuntare la casella add to PATH:
https://www.python.org/downloads/
Far partire il terminale di python e trovare il percorso completo usando questa stringa:
import sys
print(sys.executable)
Questo dovrebbe restituire una cosa del genere: C:\Users\TuoUtente\AppData\Local\Programs\Python\Python310\python.exe
Incollare questo percorso completo nel terminale (non quello di python, quello proprio del computer).
Se si usa powershell, il percorso dovrebbe essere scritto in quest'altro modo: 
& "C:\Users\daniele.angelone\AppData\Local\Programs\Python\Python313\python.exe" -m pip install fastapi uvicorn requests
